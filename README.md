## How to run | :hourglass:

#### 1. clone the repository

```bash
git clone https://github.com/johelder/login-google.git
```

#### 2. install dependencies

```bash
yarn
```

#### 3. run metro

```bash
yarn start
```

#### 4. install the app on your device/virtual device

```bash
yarn android
```
